## 🎬 Turn multiple iPhones into a pocket-sized multi-camera film system.
Synchronize multiple iPhones to capture the same moment — then cut between angles like a film director and export it as one cinematic video.

---

## 🇯🇵 日本語

複数のiPhoneを同期撮影し、映画のようにアングルを切り替えて、一本の映像として書き出す。
ポケットの中で実現する、マルチカメラ映画制作システム。
