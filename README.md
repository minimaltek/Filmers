# CineCam

## 🎬 Turn multiple iPhones into a pocket-sized multi-camera film system.

Capture the same moment from multiple synced iPhones —  
cut between angles like a film director —  
and export it as one cinematic video.

---

## 🇯🇵 日本語

複数のiPhoneを同期撮影し、映画のようにアングルを切り替えて、一本の映像として書き出す。
ポケットの中で実現する、マルチカメラ映画制作システム。

---

## ✨ Key Features

- Multi-device synchronization  
- Master / Slave recording control  
- Real-time connection monitoring  
- Cinematic multi-angle editing  
- Single unified video export
